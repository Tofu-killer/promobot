# PromoBot

LAN-deployed multi-platform AI operations automation system.

## Current Status

This repository currently stores the validated product design spec for PromoBot. Implementation will follow in this repository, and GitHub is the source of truth for future docs and code.

## Key Product Shape

- Runs on host `121`
- Accessible from the LAN through a browser
- Single-operator admin console
- Multi-project support
- Backend-only AI integration against an OpenAI-compatible endpoint
- API-first publishing with Playwright fallback where needed
